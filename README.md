# Quack-UI

> CH-UI fork for DuckDB / Quackpipe

![Quack-UI-Workspace-ezgif com-crop](https://github.com/user-attachments/assets/aec14c53-3bfb-4e1e-acb9-fcb9001c433e)


### Build from Source

#### Prerequisites
- Node.js >= 20.x
- npm >= 10.x

#### Installation Steps
Build and Test locally
```bash
npm install
npm run build
npm run dev
```
For production
```bash
npm run preview
```
