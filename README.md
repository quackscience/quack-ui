# Quack-UI

> CH-UI fork for DuckDB / Quackpipe

### Option 2: Build from Source

#### Prerequisites
- Node.js >= 20.x
- npm >= 10.x

#### Installation Steps
Build and Test locally
```bash
npm install
npm run build
npm run dev
```
For production
```bash
npm run preview
```
