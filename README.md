# Quack-UI

> CH-UI fork for DuckDB / Quackpipe

![image](https://github.com/user-attachments/assets/c0be2ae3-041e-4140-8b21-cc16d203564f)


### Build from Source

#### Prerequisites
- Node.js >= 20.x
- npm >= 10.x

#### Installation Steps
Build and Test locally
```bash
npm install
npm run build
npm run dev
```
For production
```bash
npm run preview
```
